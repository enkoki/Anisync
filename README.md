## Anisync
A simple Next.js project for learning Next.js and practicing team collaboration.
## Getting Started
### Frontend Part

First, run the development server:

```bash
cd frontend/
npm install
```
```
npm run dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
